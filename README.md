# 🔍 JobMatch ATS Pro v6

### **Motor de Matching Inteligente para Perfiles de Ingeniería**

JobMatch es una herramienta de **IA y Procesamiento de Lenguaje Natural (NLP)** diseñada para cerrar la brecha entre las Hojas de Vida técnicas y las vacantes de infraestructura en Colombia. El sistema analiza semánticamente el contenido de una HV (PDF) y lo compara con los requisitos de una vacante.

---

## 🧠 Core Técnico
- **TF-IDF Vectorization:** Ponderación de términos técnicos (SAP, SIIGO, ArcGIS).
- **Similitud de Coseno:** Cálculo matemático de compatibilidad de perfiles.
- **Consultor IA:** Sugerencias automáticas de palabras clave faltantes.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python 3.x
- **Librerías:** Scikit-learn, PyPDF2, Tkinter.
- **Entorno:** Neovim / Terminal (Linux & Windows 11).
